面向对象

面向对象的三个特征：封装，继承，多态

1.封装

1.1 封装的原则，将类的某些信息隐藏在内部，不允许外部程序直接访问，而是通过该类提供的方法（get/set）来实现对隐藏信息的操作和访问成员变量(private)

1.2 封装的好处

1.2.1 通过方法控制成员变量的操作，提高代码安全性

1.2.2 把代码用方法进行封装，提高了代码的复用性

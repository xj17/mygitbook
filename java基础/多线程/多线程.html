创建线程的两种方法

1.继承Thread类

package concurrent;
//继承Thread类
public class MyThread1 extends Thread{
    @Override
    public void run(){}
    public static void main(String[] args) {
        MyThread1 mt=new MyThread1();
        mt.start();
    }
}

2.实现Runnable接口

package concurrent;
//继承Thread类
public class MyThread1 extends Thread{
    @Override
    public void run(){}
    public static void main(String[] args) {
        MyThread1 mt=new MyThread1();
        mt.start();
    }
}

比较:Runnable接口可以被多个线程共享，适用于多个线程处理同一资源的情况，通过这种方法可以避免Thread由于java单继承特性带来的缺陷。

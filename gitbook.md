### gitbook的安装部署

#### gitbook安装

1.安装node.js 

https://nodejs.org/zh-cn/

2.验证node.js是否安装成功

node -v

npm -v

3.安装git

https://git-scm.com/

4.使用npm安装gitbook

npm install gitbook-cli -g

5.gitbook初始化

gitbook init

6.本地启动

gitbook serve

#### gitbook部署

1.将_book文件夹下的gitbook文件夹复制到根目录

2.需要在上传时将_book文件夹忽略

3.发布到<u>GitHub Pages</u>上
